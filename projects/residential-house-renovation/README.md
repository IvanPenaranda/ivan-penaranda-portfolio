# 🏠 Residential House Renovation

## Project Overview

This project involved the renovation of an existing residential house using AutoCAD.

The scope included modifying the existing layout by demolishing an internal wall and constructing a new wall to improve the functionality of the house.

---

## Project Deliverables

- Floor Plan
- Site Plan
- South Elevation
- West Elevation
- Section AA'
- Isometric Slab Detail
- Door Schedule
- Window Schedule
