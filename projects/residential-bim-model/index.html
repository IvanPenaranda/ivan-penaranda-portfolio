3D Residential BIM Model

Complete residential BIM model developed in Autodesk Revit, including architectural modelling, furnished floor plans, elevations, building sections, schedules and coordinated 3D views.

Project Type: Residential Architectural BIM Model
Software: Autodesk Revit
Scope: 3D BIM Modelling and Documentation
Deliverables: 3D Model, Plans, Elevations, Sections and PDF Drawing Set

Project Overview

This project involved creating a complete 3D residential Building Information Model (BIM) in Autodesk Revit from architectural documentation. The model includes walls, floors, roofs, doors, windows, furniture, sections, elevations, and coordinated 3D views. The final deliverables included a complete drawing package exported as PDF.
