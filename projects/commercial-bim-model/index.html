3D Commercial BIM Model

Complete commercial structural BIM model developed in Autodesk Revit from 2D drawings, including foundations, columns, beams, slabs, stair core and BIM coordination with clash detection.

Project Type: Commercial Structural BIM Model
Software: Autodesk Revit 2026
Scope: 3D Structural Modelling and BIM Coordination
Deliverables: BIM Model, Structural Views and Clash Detection

Project Overview

This project involved developing a complete multi-storey commercial structural BIM model in Autodesk Revit based on supplied 2D engineering drawings. The model includes reinforced concrete footings, columns, beams, suspended slabs and a central stair core across multiple levels. A BIM clash detection process was also completed to identify and resolve modelling conflicts, improving coordination and construction accuracy before documentation.

Project Highlights

2D to 3D BIM Development

Converted architectural 2D drawings into a fully coordinated structural BIM model using Autodesk Revit.

Structural Modelling

Modelled reinforced concrete footings, columns, beams, suspended slabs and stair core for a multi-storey commercial building.

BIM Coordination

Performed clash detection using Revit Interference Check to identify coordination issues between structural and architectural models.
