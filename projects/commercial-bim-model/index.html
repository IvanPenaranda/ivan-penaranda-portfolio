3D Commercial BIM Model

Complete commercial structural BIM model developed in Autodesk Revit from 2D drawings, including foundations, columns, beams, slabs, stair core and BIM coordination with clash detection.

Project Type: Commercial Structural BIM
Software: Autodesk Revit 2026
Location: Perth, Western Australia
Deliverables: 3D BIM Model, Structural Model, Clash Detection Report

Project Overview

This project involved converting a complete set of 2D commercial building drawings into a fully coordinated 3D structural BIM model using Autodesk Revit. The structural model includes reinforced concrete footings, columns, beams, suspended floor slabs and the complete stair core. After modelling, BIM coordination was performed using Revit Interference Check to identify clashes between structural and architectural components.
